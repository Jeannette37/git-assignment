How I will use Git and VS Code for this assignment:
I will use VS Code as my text editor to create and write the files for this assignment (like this README.md file and feature.txt). I will use the terminal to run Git commands to track the changes I make in VS Code. Git will act as my local history book, allowing me to save checkpoints (git commit), create alternate timelines (git branch), and combine them (git merge).

What this assignment demonstrates about GitHub:
This assignment demonstrates that GitHub is the cloud-based platform where we upload our local Git history. It shows how to connect a local folder to a remote repository (git remote add and git push). More importantly, it demonstrates GitHub's collaboration tools—specifically how developers use Pull Requests, Issues, and code reviews to work together on the same codebase safely.
